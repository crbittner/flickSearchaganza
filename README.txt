Formedix coding assignment
"Flickr Searchaganza"

This project was developed as an exercise for the purpose of judging the skill of the developer during the application process to the job of Front End Developer at Formedix. The project uses the Flickr API to display public photos in a web app.


Prerequisites

The only dependency is "jquery-3.2.1.min.js" which is included with the files in the scripts folder.

Installing

copy the files contained to a folder on your computer. Open "index.html" in any modern browser.


Built With

Sublime Text 3 - used to write the code
